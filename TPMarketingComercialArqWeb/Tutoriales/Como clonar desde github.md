# Qué?
Esta página explica cómo clonar el proyecto del TP desde GitHub, para que puedas tener una copia local del código y trabajar en él.

## Cómo?
Este tutorial asume que ya tenés una cuenta de GitHub y que tenés instalado Git en tu computadora, adicionalmente, es recomendable usar una interfaz gráfica para Git, como GitHub Desktop o la web, aunque también podés usar la línea de comandos si te sentís cómodo con ella. Y, por último es necesario que tengas github integrado con tu Visual Studio 2026, para poder abrir el proyecto directamente desde el IDE.

La primera vez que abras el proyecto, lo tenés que hacer de la siguiente manera. Abrí Visual Studio 2026, en la pantalla de inicio, seleccioná "Clonar un repositorio" (Clone a repository). En el campo "URL del repositorio" (Repository URL), ingresá la URL del repositorio del TP en GitHub. Esta URL se puede encontrar en la página del repositorio, haciendo clic en el botón "Code" y copiando la URL que aparece allí. https://github.com/tomg97/TPMarketingComercialArqWeb. Esto te traerá el código del proyecto a tu computadora, creando una carpeta con el nombre del repositorio, apuntado a la rama master por defecto.

Ya las siguientes veces que quieras abrir el proyecto, lo podés hacer directamente desde Visual Studio 2026, seleccionando "Abrir un proyecto o una solución" (Open a project or a solution) y navegando hasta la carpeta donde clonaste el repositorio. Allí, seleccioná el archivo de solución (.sln) para abrir el proyecto completo en Visual Studio.

## Para qué?
Para que todos podamos trabajar con el proyecto de manera colaborativa, teniendo acceso al código fuente y pudiendo hacer cambios y mejoras sin afectar el trabajo de los demás. Además, clonar el repositorio nos permite tener una copia local del proyecto, lo que facilita el desarrollo y la depuración.
