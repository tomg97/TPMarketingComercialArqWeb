# Qué?
Esta página explica como correr el proyecto del TP

## Cómo?
Primero, seguir los pasos indicados en este video https://www.youtube.com/watch?v=DobmV5-cUX0 para poder configurar tu Visual Studio 2026 y que sea compatible con proyectos .net winforms.

Segundo, abrir el proyecto con Visual Studio 2026 y correrlo con F5 o el botón de "Iniciar depuración" (Start Debugging) en la barra de herramientas. Esto compilará el proyecto y lo ejecutará en modo de depuración, permitiéndote probar la aplicación y detectar posibles errores.

Alternativamente, podés correr una página individual del proyecto, haciendo click derecho en el archivo .aspx que desees. Esto nos brinda varias opciones, de las cuales nos interesan:
- Ver en navegador, que la abre en el navegador predeterminado. Esta es la experiencia más cercana a la que tendrá el usuario final.
- Ver en el diseñador. Esta vista no es muy recomendable, ya que a menudo no muestra la página correctamente, pero puede ser útil para hacer cambios rápidos en el diseño sin tener que compilar todo el proyecto.

## Para qué?
Para que todos podamos trabajar más cómodos con el proyecto
